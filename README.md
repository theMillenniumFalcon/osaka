# Osaka
A working agent that can read, write, and edit files through natural conversation with Claude.

## Features
- File Operations: Read, list, and edit files
- Search: Search across multiple files with regex support
- Multi-file Edit: Batch edit multiple files at once
- Command Execution: Run shell commands safely
- Undo: Revert file changes
- Backup: Automatic backup before editing

## Available Commands
- Type your request naturally to the AI agent
- exit or quit - End the conversation
- undo - Revert the last file change

## Safety Features
- Automatic file backups before editing
- Command validation to prevent dangerous operations
- Edit history tracking
